## Session themes

1. Normalizer rule added for Word.Field casing (after VBA IDE
   demoted "Dim fld As Word.Field" to "Dim fld As Word.field"
   in basStyleInspector). Run-once across src/ fixed 4 cases
   across 3 files.
2. AuditHyperlinkStyling extended to check Font.Name=Carlito and
   Font.Size=9 in addition to color + underline. Caught a
   size=11 anomaly on a hyperlink inside an AuthorListItemTab
   paragraph - paragraph inherits its size, built-in Hyperlink
   style doesn't pin font/size.
3. Architectural decision: replace use of Word's built-in
   Hyperlink character style with a custom "BookHyperlink"
   style that pins all four font properties explicitly. Built-in
   Hyperlink cannot enforce font/size (inherits from paragraph);
   custom style is fully self-contained.
4. BookHyperlink implementation:
   - DefineBookHyperlinkStyle in basFixDocxRoutines.
   - Added to approved array; AuditOneStyle row added.
   - LockBookHyperlinks replaces LockHyperlinksToPalette
     (three-step: migrate built-in -> BookHyperlink, unlink
     active Hyperlinks, force-apply properties on all
     BookHyperlink runs). Built-in Hyperlink and FollowedHyperlink
     style definitions deliberately NOT touched.
   - LockHyperlinksAlwaysBlue alias removed (was due).
   - AuditHyperlinkStyling renamed AuditBookHyperlinkStyling.
   - EDSG/01-styles.md "Companion rule" rewritten; per-installation
     "Disable URL auto-format" recommendation added.
5. Test 17 (CountActiveHyperlinks) unchanged - still measures
   total Hyperlinks collection count across stories. Expected 0.
