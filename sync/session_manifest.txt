[IMPORT]  src/basBiblePalette.bas (NEW)
          - Single source of truth for named colors.
          - Each entry is a nested Scripting.Dictionary with keys
            Name, R, G, B, RgbLong, HexCode, Usage. (Public Type
            UDT not used: VBA forbids passing UDTs declared in
            .bas modules to late-bound functions; nested Dict
            sidesteps the limit while staying late-bound.)
          - GetPalette(theme) -> outer Dictionary keyed by Name.
            Only theme="Default" populated; "Dark"/"Colorblind"
            raise "not implemented".
          - Helpers: ColorFromName, NameFromColor, LongToHex,
            HexToLong, LongToRgbString, DumpPalette.
          - 12 colors: Black, White, Red, DarkRed, Green,
            DarkGreen, Emerald, Blue, Gold, Orange, Purple, Gray.
          - wdColorAutomatic deliberately excluded (sentinel, not
            a color). Office ObjectThemeColor deliberately excluded.
          - Late binding (Scripting.Dictionary via CreateObject).
            No project references added.
          - DescribeFirstRunOfColor(rgbLong) added for item 11
            diagnostics: finds first explicit-color run in any
            primary StoryRange and prints page / paragraph style /
            run style / text / context. Scope: explicit run-level
            overrides only (Find with .Font.Color matches direct
            overrides, NOT style-inherited colors).
          - DescribeStylesCarryingColor(rgbLong) added as the
            complement: walks ActiveDocument.Styles and reports
            any style whose Font.Color matches. Use this to find
            colors that the histogram counts but Find cannot
            locate (i.e., colors applied via style chain).
          - CountRunsWithColor(rgbLong) As Long: Find-based scan
            across all primary StoryRanges. Authoritative
            per-color total. Use when the Word-level histogram is
            undercounting single-char colored runs.
          - ReportRunsWithColor(rgbLong): same scan with per-story
            breakdown printed to Immediate.
          - ListRunsOfColorByStyle(rgbLong): same scan, groups runs
            by their character-style name. Use this to identify
            strays - the legitimate styles appear with large counts,
            an outlier style with count 1 (or small N) is a stray
            candidate.

[IMPORT]  src/basStyleInspector.bas
          - AuditFootnoteReferenceMarkers() As Long: walks
            ActiveDocument.Footnotes, counts FR-styled characters
            inside each footnote.Range, flags footnotes with count
            != 1. Surfaced by item 11 (Blue FR count was 2001 vs
            expected 2000). Returns anomaly count; expected 0 in
            a clean doc.
          - AuditHyperlinkStyling() As Long: walks all StoryRanges,
            Find runs styled "Hyperlink", verifies Font.Color =
            palette DarkBlue and Underline = single. Covers ALL
            Hyperlink-styled runs including REF/HYPERLINK field-
            result runs not in the Hyperlinks collection. Returns 0
            after LockHyperlinksToPalette; non-zero means a pasted
            hyperlink-styled run escaped the convention.
          - ReportHyperlinkStoryDistribution: diagnostic - per
            StoryRange, prints Hyperlinks.Count vs Hyperlink-styled-
            run count. The gap shows field-result-style hyperlinks
            that the collection misses.
