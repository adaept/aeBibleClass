# Bible Book Abbreviation Reference (Non-SBL)

This reference provides a unified, deduplicated list of the most common non-SBL Bible book abbreviations used across:

- Traditional English publishing (KJV-lineage, commentaries, devotional works)
- Standard church/academic abbreviations
- Digital shortest-form systems (Logos-style, concordances, BibleStudyTools)

## Purpose

To provide a single authoritative abbreviation set that can be used for:

- Parsing Scripture references
- Normalizing user-generated citations
- Building resolver dictionaries
- Bible-publishing QA workflows
- Cross-reference engines
- Automated macro-driven diagnostics

## Benefits

- **Deduplicated:** trailing-period variants removed (e.g., `Heb.` -> `Heb`)
- **Normalized:** shortest-form abbreviations capitalized (e.g., `1pe` -> `1Pe`)
- **Merged:** all major non-SBL systems combined
- **Stable:** safe for dictionary keys, regex patterns, and automated parsing
- **Publisher-grade:** consistent, predictable, and typographically clean
